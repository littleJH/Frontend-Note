# 构造器模式

构造器模式是：

从手动创建对象实例

```jsx
const liLei = {
    name: '李雷',
    age: 25,
    career: 'coder',
}

const hanMeiMei = {
    name: '韩梅梅',
    age: 24,
    career: 'product manager'
}
```

到通过构造函数抽象**每个对象的变与不变**

```jsx
function User(name , age, career) {
    this.name = name
    this.age = age
    this.career = career 
}

const user = new User(name, age, career)
```
